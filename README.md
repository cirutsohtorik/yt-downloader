# YT Downloader

A modern HBO Max styled YouTube video/audio downloader built with Flask and yt-dlp.

## Features

- Download videos in MP4 format
- Download audio in MP3 format
- Dynamic quality selection
- Video/audio cutting support
- Automatic thumbnail preview
- Automatic video title fetching
- HBO Max inspired UI
- Real-time video info loading
- Custom output filenames

## Tech Stack

- Python
- Flask
- yt-dlp
- FFmpeg
- HTML/CSS/JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
